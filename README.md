# most-used-words
Aplicação Desktop com JavaScript, Electron JS e Vue JS que consome os arquivos de legenda e faz o cálculo de palavras mais usadas.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

npm run serve
